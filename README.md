# Hello

Hello is a chat application in which user can enter their names and start chatting. This app uses Web-socket to feed messages in real-time. Backend is build using Spring boot.

## Technologies used: 
  
  -Spring Boot
  -Spring Stomp
  -SockJS
  -JavaScript
  -Bootstrap 5
